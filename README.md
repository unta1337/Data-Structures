# C를 이용해 구현된 자료구조  

## C에서 사용할 수 있는 ADT의 구현  
* 리스트, 큐, 스택 등, C에서 제공하지 않는 자료구조에 대한 구현이다.  
* 구현하는 ADT와 그 연산은 C++의 STL을 참고했다.  
* 포인터와 동적할당을 활용해 자료구조 구현한다.  
* 개발 단계이므로 개별 함수에 대한 이름이나 매개변수 순서 등은 언제든지 변경될 수 있다.
